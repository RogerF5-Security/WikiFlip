# WikiFlip

WikiFlip is an offline cybersecurity dictionary for Flipper Zero.

It provides a categorized reference for pentesting, blue team operations, standards, protocols, hardware hacking, threat intelligence, CVEs, and OWASP Top 10:2025.

English is the default language. Spanish is available from the in-app Settings menu. The app works fully offline and stores all entries as static C data, so it does not require microSD parsing, network access, or external files at runtime.

## Features

- 100 cybersecurity terms
- Matching Spanish term set available from Settings
- Full OWASP Top 10:2025 category coverage
- English UI and definitions by default
- In-app language selector
- Native Flipper Zero submenu navigation
- Scrollable long-form definitions
- Offline static dictionary
- Compatible with official firmware, Momentum, and FBT/uFBT-compatible forks
